## Tech Stack

TypeScript, React, Tailwind, React Router


## Run Locally

Clone the project

```bash
  git clone git@github.com:denysdev1/react-tailwind-components-page.git
```

Go to the project directory

```bash
  cd react-tailwind-components-page
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
